Hai! Apakah kamu ingin membuat bot telegram dengan mtproto dalam javascript atau typescript? Jika iya, tgsnake adalah framework yang terbaik untukmu. Secara singkat, tgsnake adalah sebuah framework mtproto yang menggunakan gramjs sebagai base nya. Kami telah menyederhanakan Api mentah gramjs kedalam method yang simpel dan mudah.
## Mendapatkan ApiHash dan ApiId
Dimana saya mendapatkan apiHash dan apiId?  
Kamu bisa mendapatkannya dari [my.telegram.org](https://my.telegram.org). Kamu harus membuat aplikasi disana. Peringatan! Jangan sebarkan ApiHash dan ApiId kamu!
## Setelah membuat aplikasi
Setelah kamu membuat aplikasi, kamu membuat proyek di komputermu. Kemudian install framework `tgsnake`.
1. `mkdir tgsnake-project`
2. `cd tgsnake-project`
3. `npm init -y`
4. `npm i tgsnake --save`
5. `code .`   
   Kemudian buat file `index.js` dan mulai menuliskan kode. Sebagai contoh kamu bisa lihat di [halaman awal](/). Kemudian jalankan kodemu.
## Bagaimana cara menghasilkan stringSession?
Untuk menghasilkan stringSession kamu bisa menggunakan method `generateSession`.

Untuk pertanyaan lebih lanjut, kamu bisa bertanya di grup atau pm saya di telegram.
