Untuk menangani update kamu bisa menggunakan method `hears`,`command`,`use`,`on`. Sebagai contoh kamu bisa lihat ini :

## Menggunakan `hears`
Method ini mengizinkan kamu untuk menggunkana regex atau kamu bisa mengisi dengan beberapa text

<TabGroup>
    <TabContent name="javascript">
    ```ts
    // index.js
    const {Snake} = require("tgsnake")
    const bot = new Snake() // isian opsi. kamu bisa menggunakan file tgsnake.config.js.
    // dengan teks
    bot.hears("hello",(ctx)=>{
      ctx.reply("Hi") 
      if(ctx.replyToMessage){ 
        ctx.replyToMessage.reply("Who are you?")
      }
    })
    // dengan regex 
    bot.hears(/hello/i,(ctx,match)=>{
      ctx.reply("Hi") 
      if(ctx.replyToMessage){ 
        ctx.replyToMessage.reply("Who are you?")
      }
    })
    bot.run() // jalankan 🐍
    ```
    </TabContent>
    <TabContent name="typescript">

## Menggunakan `command`
Method ini hanya mengizinkanmu menggunakan string atau array string. Awalannya `.!/`

<TabGroup>
    <TabContent name="javascript">
    ```ts
    // index.js
    const {Snake} = require("tgsnake")
    const bot = new Snake() // isian opsi. kamu bisa menggunakan file tgsnake.config.js.
    // dengan teks
    bot.command("hello",(ctx)=>{
      ctx.reply("Hi") 
      if(ctx.replyToMessage){ 
        ctx.replyToMessage.reply("Who are you?")
      }
    })
    // dengan array
    bot.command(["hello","hi"],(ctx)=>{
      ctx.reply("Hi") 
      if(ctx.replyToMessage){ 
        ctx.replyToMessage.reply("Who are you?")
      }
    })
    bot.run() // jalankan 🐍
    ```
    </TabContent>
    <TabContent name="typescript">

Kemudian kirim pesan `!hello` atau `!hi`
## Menggunakan `use`
Ini perangkat tengah. Fungsi ini akan di jalankan lebih dahulu sebelum yang lain.

<TabGroup>
    <TabContent name="javascript">
    ```ts 
    // index.js
    const {Snake} = require("tgsnake")
    const bot = new Snake() 
    bot.use((ctx,next)=>{
        console.log(ctx)
        return next()
    })
    bot.run() // jalankan 🐍
    ```
    </TabContent>
    <TabContent name="typescript">
    ```ts 
    // index.ts
    import {Snake} from "tgsnake"
    const bot = new Snake() 
    bot.use((ctx,next)=>{
        console.log(ctx)
        return next()
    })
    bot.run() // jalankan 🐍
    ```
    </TabContent>
</TabGroup>
Kamu dapat menggunakan `Composer` sebagai perangkat tengah.
<TabGroup>
    <TabContent name="javascript">
    ```ts 
    // index.js
    const {Snake,Composer} = require("tgsnake")
    const bot = new Snake() 
    const startComposer = new Composer() 
    startComposer.command("start",(ctx)=>{
        ctx.reply("Welcome")
    })
    bot.use(startComposer.middleware())
    bot.run() // jalankan 🐍
    ```
    </TabContent>
    <TabContent name="typescript">
    ```ts 
    // index.ts
    import {Snake,Composer} from "tgsnake"
    ```

</TabGroup>
