Ini adalah file konfigurasi untuk tgsnake. Kamu bisa melewati semua opsi di sana agar file indeks kamu bersih.

## Opsi
Kita memiliki banyak opsi jika kamu menggunakannya bersamaan ini akan membuat file indeksmu panjang. Opsinya diantaranya:
|        nama         |      tipe      |                                                                                                                                                              deskripsi                                                                                                                                                              |
|:-------------------:|:--------------:|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------:|
|       logger        |     string     |                                                                                                                                       Mengatur level logging untuk gramjs. Defaultnya "none".                                                                                                                                       |
|       apiHash       |     string     |                                                                                                                                Sebuah api_hash yang didapat dari [my.telegram.org](my.telegram.org)                                                                                                                                 |
|        apiId        |     number     |                                                                                                                                 Sebuah api_id yang didapat dari [my.telegram.org](my.telegram.org)                                                                                                                                  |
|       session       |     string     |                                                                                                                                                             Sesi string                                                                                                                                                             |
|      botToken       |     string     |                                                                                                                             Token Bot dari botFather. Ini diperlukan jika kamu butuh login sebagai bot                                                                                                                              |
|     tgSnakeLog      |    boolean     |                                                                                                              Jika diatur, tgsnake akan menampilkan pesan di konsol seperti selamat datang di tgsnake atau yang lainnya                                                                                                              |
|     sessionName     |     string     |                                                                                                                                                    Untuk menyimpan sesi string.                                                                                                                                                     |
|    storeSession     |    boolean     |                                                                                                                                                      Menggunakan store session                                                                                                                                                      |
|     connection      |      any       |                                                                             Instansi koneksi yang akan digunakan saat membuat konekasi baru ke server. Harus berupa type. Defaultnya ConnectionTCPFull di Node dan ConnectionTCPObfuscated di browser.                                                                              |
|       useIPV6       |    boolean     |                                                                                                                            Apakan hubungkan ke server melalui IPv6 atau tidak. Secara default ini false.                                                                                                                            |
|       timeout       |     number     |                                                                                                               Batas waktu dalam detik untuk digunakan saat menghubungkan. Permintaan di coba lagi ketika Telegram mengalami masalah internal (karena error INTERNAL atau error RPC_CALL_FAIL), ketika ada errors.FloodWaitError yang lebih kecil dari floodSleepThresold, atau ketika ada kesalahan migrasi. defaultnya 5.                  |

|  sequentialUpdates  |    boolean     |                                                                                                                                                tidak melakukan apapun untuk sekarang                                                                                                                                                |
| floodSleepThreshold |     number     | Ambang dibawah yang mana library harus sleep otomatis saat flood wait dan error slow mode wait (termasuk). Sebagai contoh, jika FloodWaitError terjadi selama 17d dan floodSleepThresold 20d, library akan sleep secara otomatis. Jika errornya selama 21d, akan me-raise FloodWaitError sebagai penggantinya. defaultnya 60 detik. |
|     deviceModel     |     string     |                                                                                                                   Model perangkat yang akan dikirim ketika membuat koneksi awal. Defaultnya os.type().toString().                                                                                                                   |
|    systemVersion    |     string     |                                                                                                                  Versi sistem yang akan dikirim ketika membuat koneksi awal. defaultnya os.release().toString() -.                                                                                                                  |
|     appVersion      |     string     |                                                                                                                            Versi aplikasi yang akan dikirim ketika membuat koneksi awal. Defaultnya 1.0.                                                                                                                            |

Defaultnya 'en'.                                                                                                                             |

Defaultnya 'en'.                                                                                                                          |

## Contoh Penggunaan
Pertama kamu harus membuat `tgsnake.config.js` di direktory root (dimana package.json tersedia).   
Kemudian salin kode ini
```js
// tgsnake.config.js
module.exports = {
    apiHash : "api hash kamu" 
    // isi opsi yang tersedia disini.
}
```

Kemudian file indeks seharusnya :
