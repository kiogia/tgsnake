## Gambaran
Mengirim lokasi file Dokumen/url/buffer.
## Parameter
```ts
ctx.telegram.sendDocument(chatId,document,more)
```
- `chatId` (_angka|string_)
     - Obrolan/Grup/Id Saluran.
- `document` (_string|Buffer_)
     - Lokasi File/Url/Buffer
- `more` (_[sendMediaMoreParams](/en/telegram/media/SendMedia#parameters)_)
## Contoh
```ts highlight= 2
bot.command("doc",async (ctx) => {
     let hasil = await ctx.telegram.sendDocument(ctx.chat.id,"https://tgsnake.js.org/images/tgsnake.jpg")
})
```
## Kemungkinan Kesalahan
Sama seperti [sendMedia](/methods/Media/sendMedia#possibleerror)
