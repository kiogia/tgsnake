## Gambaran
Media pengirim pesan.
## Parameter
```ts
ctx.telegram.sendMedia(chatId,media,more)
```
- `chatId` (_number|string|bigint_)
    - Obrolan/Grup/Id Saluran.
- `media` (_[Api.TypeInputMedia](https://gram.js.org/beta/modules/tl_api.api.html#typeinputmedia)_)
    - Media Pesan.
- `more` (_objek_)
    - `silent` (_boolean_)
        - kirim pesan tanpa pemberitahuan.
    - `background` (_boolean_)
        - kirim sebagai pesan latar belakang.
    - `parseMode` (_string_)
        - mode parse untuk mengatur gaya teks.
    - `clearDraft` (_boolean_)
        - kosongkan bidang draf.
    - `replyToMsgId` (_number_)
        - kirim dengan membalas pesan.
    - `replyMarkup` (_TypeReplyMarkup_)
        - Markup balasan untuk mengirim tombol bot.
    - `entities` (_Entities[]_)
        - entitas pesan untuk memberi gaya pada teks.
    - `scheduleDate` (_number_)
        - membuat pesan jadwal.
    - `caption` (_string_)
        - keterangan media.
    - `workers` (_number_)
        - pekerja untuk mengunggah media.
