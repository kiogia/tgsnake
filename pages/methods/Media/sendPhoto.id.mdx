## Gambaran
Mengirim foto dengan fileId/lokasi file/url/buffer.
## Parameter
```ts
ctx.telegram.sendPhoto(chatId,photo,more)
```
- `chatId` (_number|string|bigint_)
     - Obrolan/Grup/Id Saluran.
- `photo` (_string|Buffer|Media_)
     - FileId/Lokasi File/Url/Buffer
- `more` (_[sendMediaMoreParams](/en/telegram/media/SendMedia#parameters)_)
## Contoh
```ts highlights= 3
bot.on("message",async (ctx) => {
     if(ctx.media && ctx.media.type == "foto"){
         let hasil = await ctx.telegram.sendPhoto(ctx.chat.id,ctx.media.fileId)
         console.log(hasil)
     }
})
```
## Kemungkinan Kesalahan
Sama seperti [sendMedia](/methods/Media/sendMedia#possibleerror)
