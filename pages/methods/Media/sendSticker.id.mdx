## Gambaran
Mengirim stiker dengan fileId/lokasi file/url/buffer.
## Parameter
```ts
ctx.telegram.sendSticker(chatId,sticker)
```
- `chatId` (_number|string|bigint_)
     - Obrolan/Grup/Id Saluran.
- `sticker` (_string|Buffer_).
     - File jalur/FileId/Buffer.
## Contoh
```ts highlight=3
bot.on("message",async (ctx) => {
     if(ctx.media && ctx.media.type == "sticker"){
         let hasil = await ctx.telegram.sendSticker(ctx.chat.id,ctx.media.fileId)
         console.log(hasil)
     }
})
```
## Kemungkinan Kesalahan
Sama seperti [sendMedia](/methods/Media/sendMedia#possibleerror)
