## Gambaran
Dapatkan log admin saluran/supergrup
## Parameter
```ts
ctx.telegram.getAdminLog(chatId, more)
```
- `chatId` (_number|string|bigint_)
    - Id Obrolan/Saluran/Grup.
- `more` (_Object_)
    - `q` (_string_)
        - Permintaan pencarian, boleh kosong.
    - `maxId` (_bigint_)
        - ID maksimum pesan yang akan dikembalikan. _(>=1)_
    - `minId` (_bigint_)
        - Minimum ID pesan untuk dikembalikan. _(>=1)_
    - `limit` (_number_)
        - Jumlah maksimum hasil untuk kembali. _(>=1)_
          
    _di bawah ini adalah filter yang tersedia. Anda dapat menemukan penjelasannya [di sini](https://core.telegram.org/constructor/channelAdminLogEventsFilter)._
    - `join` (_boolean_) 
    - `leave` (_boolean_)
    - `invite` (_boolean_)
    - `ban` (_boolean_)
    - `unban` (_boolean_)
    - `kick` (_boolean_)
    - `unkick` (_boolean_)
    - `promote` (_boolean_)
    - `demote` (_boolean_)
    - `info` (_boolean_)
    - `settings` (_boolean_)
    - `pinned` (_boolean_)
    - `groupCall` (_boolean_)
    - `invites` (_boolean_)
    - `edit` (_boolean_)
    - `delete` (_boolean_)
## Contoh
```ts highlight= 3
bot.command("getAdminLog",async (ctx) => {
    if(!ctx.chat.private){
        let hasil = await ctx.telegram.getAdminLog(ctx.chat.id)
        console.log(hasil)
    }
})
```
## Kemungkinan Kesalahan

| kode | tipe | deskripsi |
| :---: | :---: | :------:|
| 400 | CHANNEL_INVALID | Saluran yang disediakan tidak valid |
| 400 | CHANNEL_PRIVATE | Anda belum bergabung dengan saluran/supergrup ini |
| 400 | CHAT_ADMIN_REQUIRED | Anda harus menjadi admin dalam obrolan ini untuk melakukan ini |
| 403 | CHAT_WRITE_FORBIDDEN | Anda tidak dapat menulis di obrolan ini |
| 400 | MSG_ID_INVALID | ID pesan yang diberikan tidak valid |
