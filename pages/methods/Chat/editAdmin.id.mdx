## Gambaran
Ubah hak admin pengguna di grup/saluran..
## Parameters 
```ts 
ctx.telegram.editAdmin(chatId,userId,more)
```
- `chatId` (_number|string|bigint_)
    - Id Obrolan/Saluran/Grup. 
- `userId` (_number|string|bigint_) 
    - User id. 
- `more` (_object_) 
    - `changeInfo` (_boolean_) 
        - Jika disetel, izinkan admin untuk mengubah deskripsi saluran/grup.
    - `postMessages` (_boolean_)
        - Jika disetel, izinkan admin memposting pesan di saluran.
    - `editMessages` (_boolean_) 
        - Jika disetel, izinkan admin juga mengedit pesan dari admin lain di saluran.
    - `deleteMessages` (_boolean_) 
        - Jika disetel, izinkan admin juga menghapus pesan dari admin lain di saluran.
    - `banUsers` (_boolean_) 
        - Jika disetel, izinkan admin untuk mengeluarkan pengguna dari saluran/grup.
    - `inviteUsers` (_boolean_) 
        - Jika disetel, izinkan admin mengundang pengguna di saluran/grup.
    - `pinMessages` (_boolean_) 
        - Jika disetel, izinkan admin untuk menyematkan pesan di saluran/grup.
    - `addAdmins` (_boolean_) 
        - Jika disetel, izinkan admin untuk menambahkan admin lain dengan izin yang sama (atau lebih terbatas) di saluran/grup.
    - `anonymous` (_boolean_) 
        - Apakah admin ini anonim.
    - `manageCall` (_boolean_) 
        - Jika disetel, izinkan admin mengelola panggilan di grup/saluran.
    - `rank` (_string_) 
        - Menunjukkan peran (peringkat) admin dalam grup: hanya string arbitrer.
## Example 
```ts highlight=3
bot.command("promote",async (ctx) => {
    if((!ctx.chat.private) && ctx.replyToMessage){
        let results = await ctx.telegram.editAdmin(ctx.chat.id,ctx.replyToMessage.from.id) 
        console.log(results)
    }
})
```
