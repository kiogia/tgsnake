## Gambaran
Melarang/membatalkan larangan/menendang pengguna di grup/saluran super.
## Parameters
```ts
ctx.telegram.editBanned(chatId,userId,more)
```
- `chatId` (_number|string|bigint_)
    - Id Obrolan/Grup/Saluran.
- `userId` (_number|string|bigint_)
    - Identitas pengguna.
- `more` (_objek_)
    - `untilDate` (_number_)
      - Validitas izin tersebut (dianggap selamanya nilai apa pun kurang dari 30 detik atau lebih dari 366 hari).
    - `viewMessages` (_boolean_)
      - Jika disetel, tidak mengizinkan pengguna melihat pesan di grup super/saluran/obrolan.
    - `sendMessage` (_boolean_)
      - Jika disetel, tidak mengizinkan pengguna mengirim pesan dalam supergrup/obrolan.
    - `sendMedia` (_boolean_)
      - Jika disetel, tidak mengizinkan pengguna mengirim media apa pun dalam supergrup/obrolan.
    - `sendStickers` (_boolean_)
      - Jika disetel, tidak mengizinkan pengguna mengirim stiker di supergrup/obrolan.
    - `sendGifs` (_boolean_)
      - Jika disetel, tidak mengizinkan pengguna mengirim gif di grup super/obrolan.
    - `sendGames` (_boolean_)
      - Jika disetel, tidak mengizinkan pengguna mengirim game dalam supergrup/obrolan.
    - `sendInline` (_boolean_)
      - Jika disetel, tidak mengizinkan pengguna untuk menggunakan bot sebaris dalam supergrup/obrolan.
    - `sendPolls` (_boolean_)
      - Jika disetel, tidak mengizinkan pengguna mengirim stiker di supergrup/obrolan.
    - `changeInfo` (_boolean_)
      - Jika disetel, tidak mengizinkan pengguna mengubah deskripsi supergrup/obrolan.
    - `inviteUsers` (_boolean_)
      - Jika disetel, tidak mengizinkan pengguna mana pun untuk mengundang pengguna di grup super/obrolan.
    - `pinMessages` (_boolean_)
      - Jika disetel, tidak mengizinkan pengguna mana pun untuk menyematkan pesan di grup super/obrolan.
    - `embedLinks` (_boolean_)
      - Jika disetel, tidak mengizinkan pengguna untuk menyematkan tautan dalam pesan grup super/obrolan.
## Contoh
```ts highlight=3
bot.command("ban",async (ctx) => {
  if((!ctx.chat.private) && ctx.replyToMessage){
   let hasil = await ctx.telegram.editBanned(ctx.chat.id,ctx.replyToMessage.from.id)
    console.log(hasil)
  }
})
```
## Kemungkinan Kesalahan

| kode | tipe | deskripsi |
| :---: | :---: | :------:|
| 400 | CHANNEL_INVALID | Saluran yang disediakan tidak valid |
| 400 | CHANNEL_PRIVATE | Anda belum bergabung dengan saluran/supergrup ini |
| 400 | CHAT_ADMIN_REQUIRED | Anda harus menjadi admin dalam obrolan ini untuk melakukan ini |
| 403 | CHAT_WRITE_FORBIDDEN | Anda tidak dapat menulis di obrolan ini |
| 400 | MSG_ID_INVALID | ID pesan yang diberikan tidak valid |
| 400 | PINNED_DIALOGS_TOO_MUCH | Terlalu banyak dialog yang dipasangi pin |
| 400 | USER_ADMIN_INVALID | Anda bukan admin |
| 400 | USER_ID_INVALID | ID pengguna yang diberikan tidak valid |
