## Gambaran
Mendapatkan daftar dari semua peserta di saluran atau obrolan.

## Parameter
```ts
ctx.telegram.getChatMembers(chatId,filters)
```

- `obrolan` (_angka|string_)
    - Obrolan atau id saluran untuk mendapatkan daftar anggota.
- `filter` (_objek_)
    - offset (_angka_)
        - Dari indeks apa ke pagination dimulai.
    - limit (_angka_)
        - Anggota daftar maksimal dalam pagination, defaultnya adalah `200` anggota.
    - query (_string_)
        - Kueri peserta berdasarkan nama
    - filter (`string`)
        - `all` Mendapatkan semua anggota.
        - `kicked` Hanya mendapatkan anggota yang ditendang.
        - `terbatas` Hanya mendapatkan anggota terbatas
        - `bots` Hanya mendapatkan bot.
        - `recents` Mendapatkan hanya anggota baru-baru ini.
        - `administrator` Hanya mendapatkan administrator.

## Contoh
```ts highlight=2
bot.command("getChatMembers",async (ctx) => {
    let result = await ctx.telegram.getParticipants(ctx.chat.id) // getChatMembers dan getParticipants adalah metode yang sama.
    console.log(hasil)
})
```
