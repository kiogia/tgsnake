## Overview 
This method allow you to edit a message. 
## Parameters 
```ts 
ctx.telegram.editMessage(chatId,messageId,text,more)
```
- `chatId` (_number|string|bigint_) 
    - Chat/Groups/Channel id.
- `messageId` (_number_)  
    - Message id to be edited.
- `text` (_string_)  
    - New Message/Caption. You can pass with blank string (`""`) if you want to edit media.
- `more` (_object_)  
    - `noWebpage` (_boolean_) 
        - Disable webpage preview
    - `media` (_[Api.TypeInputMedia](https://gram.js.org/beta/modules/tl_api.api.html#typeinputmedia)_) 
        - New attached media
    - `replyMarkup` (_TypeReplyMarkup_)
        - Reply markup for inline keyboards.
    - `entities` (_Entities[]_)
        - Message Entities
    - `scheduleDate` (_number_)
        - Scheduled message date for scheduled messages.
    - `parseMode` (_string_) 
        - Parse mode.
