## Overview 
Returns the list of messages by their IDs. 
## Parameters 
```ts 
ctx.telegram.getMessages(chatId,messageId,replies)
```
- `chatId` (_number|string|bigint_) 
    - Chat/Groups/Channel id.
- `messageId` (_number[]_) 
    - Message IDs.
- `replies` (_boolean_) 
    - if `true` it will getting the nested reply. and will making floodwait.
## Example 
```ts highlight=2
bot.command("getMessages",async (ctx)=>{
    let results = await ctx.telegram.getMessages(ctx.chat.id,[ctx.id]) 
    console.log(results) 
})
```
