## Gambaran
Meneruskan pesan dengan ID mereka.
## Parameter
```ts
ctx.telegram.forwardMessages(chatId,fromChatId,messageId,lebih banyak)
```
- `chatId` (_number|string|bigint_)
    - Tujuan.
- `fromChatId` (_number|string|bigint_)
    - Sumber pesan.
- `messageId` (_number[]_)
    - ID pesan yang akan diteruskan.
- `lebih` (_objek_)
    - `withMyScore` (_boolean_)
        - Saat meneruskan game, apakah akan memasukkan skor Anda ke dalam game.
    - `silent` (_boolean_)
        - Apakah akan mengirim pesan secara diam-diam (tidak ada pemberitahuan yang akan dipicu pada klien tujuan)
    - `background` (_boolean_)
        - Apakah akan mengirim pesan di latar belakang.
    - `scheduleDate` (_number_)
        - Tanggal pesan terjadwal untuk pesan terjadwal.
## Contoh
```ts highlight=2
bot.command("forward", async (ctx) => {
    let hasil = await ctx.telegram.forwardMessages(ctx.chat.id,ctx.chat.id,[ctx.id])
    return console.log(hasil)
})
```
