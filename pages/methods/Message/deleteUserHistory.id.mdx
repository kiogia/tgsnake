## Gambaran
Hapus semua pesan yang dikirim oleh pengguna tertentu dalam supergrup.

<Callout info="error" emoji="⚠️">
Mungkin metode ini tidak berfungsi di versi ini yang menggunakan versi gramjs `2.0.8` di mana `Api.channels.DeleteUserHistory` tidak terdefinisi atau tidak ada.
</Callout>

## Parameter
```ts
ctx.telegram.deleteUserHistory(chatId,userId)
```
- `chatId` (_number|string|bigint_)
    - id supergrup.
- `userId` (_number|string|bigint_)
    - Pengguna yang pesannya harus dihapus.

## Contoh
```ts highlight=2
bot.command("deleteMe", async (ctx) => {
    let hasil = await ctx.telegram.deleteUserHistory(ctx.chat.id,ctx.from.id)
    return console.log(hasil)
})
```
## Kemungkinan Kesalahan
| kode | ketik | deskripsi |
| :---: | :---: | :------:|
| 400 | CHANNEL_INVALID | Saluran yang disediakan tidak valid. |
| 400 | CHANNEL_PRIVATE | Anda belum bergabung dengan saluran/supergrup ini. |
| 400 | CHAT_ADMIN_REQUIRED | Anda harus menjadi admin dalam obrolan ini untuk melakukan ini. |
| 403 | CHAT_WRITE_FORBIDDEN | Anda tidak dapat menulis di obrolan ini. |
| 400 | MSG_ID_INVALID | ID pesan yang diberikan tidak valid.
| 400 | USER_ID_INVALID | ID pengguna yang diberikan tidak valid.
