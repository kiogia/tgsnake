## Gambaran
Metode ini memungkinkan Anda untuk menghapus pesan dengan pengidentifikasinya
## Parameter
```ts
ctx.telegram.deleteMessages(chatId,mesageId)
```
- `chatId` (_number|string|bigint_)
     - Pengguna atau obrolan, di mana pesan berada.
- `messageId` (_number[]_)
     - Daftar ID Pesan yang akan dihapus.
## Contoh
```ts highlight= 2
bot.command("hapus", async (ctx) => {
     let hasil = await ctx.telegram.deleteMessages(ctx.chat.id,[ctx.id])
     return console.log(hasil)
})
```
## Kemungkinan Kesalahan
| kode | ketik | deskripsi |
| :---: | :---: | :------:|
| 400 | CHANNEL_INVALID | Saluran yang disediakan tidak valid. |
| 400 | CHANNEL_PRIVATE | Anda belum bergabung dengan saluran/supergrup ini. |
| 403 | MESSAGE_DELETE_FORBIDDEN | Anda tidak dapat menghapus salah satu pesan yang Anda coba hapus, kemungkinan besar karena itu adalah pesan layanan. |
| 400 | MSG_ID_INVALID | ID pesan yang diberikan tidak valid. |
