## Gambaran
Metode ini memungkinkan Anda untuk menghapus riwayat komunikasi.
## Parameter
```ts
ctx.telegram.deleteHistory(chatId, more)
```
- `chatId` (_number|string|bigint_)
    - Pengguna atau obrolan, riwayat komunikasi yang akan dihapus.
- `more` (_objek_)
    - `maxId` (_number_)
        - ID maksimum pesan yang akan dihapus.
    - `revoke` (_boolean_)
        - Apakah akan menghapus riwayat pesan untuk semua peserta obrolan.
    - `justClear` (_boolean_)
        - Cukup hapus riwayat untuk pengguna saat ini, tanpa benar-benar menghapus pesan untuk setiap pengguna obrolan.

## Contoh
```ts highlight= 2
bot.command("hapus", async (ctx) => {
    let hasil = await ctx.telegram.deleteHistory(ctx.chat.id)
    kembali console.log(hasil)
})
```
