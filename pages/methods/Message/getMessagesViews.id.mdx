## Gambaran
Dapatkan dan tingkatkan penghitung tampilan pesan yang dikirim atau diteruskan dari saluran.
## Parameter
```ts
ctx.telegram.getMessagesViews(chatId,messageId,increment)
```
- `chatId` (_number|string|bigint_)
    - Di mana pesan itu ditemukan.
- `messageId` (_number[]_)
    - ID pesan.
- `increment` (_boolean_)
    - Apakah akan menandai pesan sebagai dilihat dan menambah penghitung tampilan
## Contoh
```ts highlight=2
bot.command("getMessagesViews",async (ctx)=>{
    let hasil = await ctx.telegram.getMessagesViews(ctx.chat.id,[ctx.id])
    console.log(hasil)
})
```
## Kemungkinan Kesalahan
| kode | ketik | deskripsi |
| :---: | :---: | :------:|
| 400 | CHANNEL_INVALID | Saluran yang disediakan tidak valid |
| 400 | CHANNEL_PRIVATE | Anda belum bergabung dengan saluran/supergrup ini |
| 400 | CHAT_ID_INVALID | Id obrolan yang diberikan tidak valid |
| 400 | MSG_ID_INVALID | ID pesan yang diberikan tidak valid |
| 400 | PEER_ID_INVALID | ID rekan yang diberikan tidak valid |
