## Overview
Sending a message text.
## Parameters
```ts 
ctx.telegram.sendMessage(chatId,text,more)
```
- `chatId` (_number|string_)
    - Destinations.
- `text` (_string_)
    - Message.
- `more` (_object_)
    - `noWebpage` (_boolean_)
        - disable web page preview.
    - `silent` (_boolean_)
        - send message with silent.
    - `background` (_boolean_)
        - send as background message.
    - `parseMode` (_string_)
        - parse mode to style the text.
    - `clearDraft` (_boolean_)
        - clear the draf field.
    - `replyToMsgId` (_number_)
        - send with reply a message.
    - `replyMarkup` (_TypeReplyMarkup_)
        - Reply markup for sending bot buttons.
    - `entities` (_Entities[]_)
        - message entities to style the text.
    - `scheduleDate` (_number_)
        - create a schedule message.
## Example
```ts highlight=2
bot.command("sendMessage",async (ctx)=>{
    let results = await ctx.telegram.sendMessage(ctx.chat.id,"hello")
    console.log(results)
})
```
