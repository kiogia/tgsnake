## Gambaran
Mendapatkan semua foto profil pengguna.
## Parameter
```ts
ctx.telegram.getUserPhotos(chatId, more)
```
- `chatId` (_number|string|bigint_)
     - id pengguna.
- `more` (_object_)
     - `offset` (_number_)
         - Jumlah elemen daftar yang akan dilewati.
     - `maxId` (_bigint_)
         - Jika nilai positif ditransfer, metode ini hanya akan mengembalikan foto dengan ID kurang dari yang ditetapkan.
     - `limit` (_number_)
         - Jumlah elemen daftar yang akan dikembalikan
## Contoh
```ts highlight=2
bot.command("getUserPhotos",async (ctx)=>{
     let hasil = await ctx.telegram.getUserPhotos(ctx.from.id)
     console.log(hasil)
})
```
## Kemungkinan Kesalahan

| kode | ketik | deskripsi |
| :---: | :---: | :------:|
| 400 | MAX_ID_INVALID | ID maks yang diberikan tidak valid |
| 400 | USER_ID_INVALID | ID pengguna yang diberikan tidak valid |
